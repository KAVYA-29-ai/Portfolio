# Kavya Rajput - AI Engineer Portfolio

A premium, modern web portfolio showcasing my professional journey, technical skills, and creative AI projects. This portfolio features advanced 3D components, dynamic animations, and a sophisticated dark theme.

## 🚀 Key Features

- **3D Interactive Carousels**: Custom-built 3D carousels for showcasing Certifications and Projects with smooth navigation and mobile responsiveness.
- **Premium UI/UX**: Dark-themed aesthetic inspired by modern engineering dashboards, featuring glassmorphism, glowing accents, and tech-vine SVG animations.
- **Dynamic Branding**: Neon-glow text effect on name headers and a consistent global tagline across all sections.
- **Project Showcase**: Detailed cards for AI research, NLP, and web development projects with direct links to live demos.
- **Responsive Layout**: Fully optimized for various screen sizes, from mobile devices to desktop monitors.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3 (Vanilla)**: Custom styling with CSS Grid, Flexbox, and Keyframe Animations.
- **JavaScript (Vanilla)**: Interactive logic for 3D carousels and navigation.
- **SVG Animations**: Circuit-inspired decorative vines and tech elements.

## 📂 Project Structure

